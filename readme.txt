== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sportreef-sports-widgets` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Do I need Sportreef account to be able to use the plugin? =

Yes. We provide free accounts with certain traffic limitations. It is also possible to upgrade to more professional account.
Please visit https://my.sportreef.com/#/register to register.

= Which widget id do I need to enter in short code and/or widgets menu? =

You need the unique widget id which can be found in your personal Sportreef space once you log in at https://my.sportreef.com.
Just look for `data-sportreef-widget` attribute when you edit a widget. Copy/paste it in your short code or widget menu.

= Which domain / host do I need to enter in my Sportreef space? =

"Full domain / Host" value has to match host of the page where you wish your widget to appear. For example if the URL of the page is:
http://www.example.com/my-sports, you would need to specify "www.example.com" as domain/host name.
